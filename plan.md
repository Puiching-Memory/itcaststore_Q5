```
├── itcaststore/
│   └── Dockerfile              # 后端应用镜像 ✅
└── .dockerignore               # Docker 忽略文件 ✅
```

#### 部署命令示例

```bash
# 构建并启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down

# 停止服务并删除数据卷
docker-compose down -v
```
